# Bluestock Mutual Fund Analytics Capstone

## Project Overview
This project provides an end-to-end analytical pipeline for evaluating mutual fund performance, risk (VaR/CVaR), and investor behavioral patterns. 

## Key Features
* **Automated ETL:** Clean, scalable data ingestion into SQLite.
* **Risk Engine:** Sortino Ratio, 95% VaR, and Rolling Sharpe analysis.
* **Behavioral Analysis:** Predictive SIP churn modeling and cohort retention tracking.

## Getting Started
1. **Clone the repo:** `git clone https://github.com/your-username/bluestock-mf-analytics.git`
2. **Install dependencies:** `pip install -r requirements.txt`
3. **Execute:** Run `python scripts/run_pipeline.py` to process data and generate reports.
